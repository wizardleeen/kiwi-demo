# How to Build

1. Clone the Repository
    ```bash
    git clone git@github.com:wizardleeen/kiwi-demo.git 
    ```
2. Navigate into the Project Directory
   ```bash
   cd kiwi-demo
   ```
3. Configure the Deployment Server
    ```bash
    kiwi host https://metavm.tech
    ```
4. Build and Deploy
    ```bash
    kiwi deploy
    ```
